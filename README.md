# README – Ansvarsfraskrivelse og Licens

## Formål
Denne quiz er udelukkende udviklet til træning og læring. Den er ikke en officiel eksamen, certificering eller garanti for beståelse af nogen form for test.

## Ansvarsfraskrivelse
- Jeg (udvikleren/udgiveren) påtager mig intet ansvar for:
  - Resultater opnået ved brug af quizzen.
  - Om brugeren består eller dumper en efterfølgende eksamen.
  - Eventuelle konsekvenser af brugen, herunder tab af tid, økonomiske omkostninger eller andre følger.
- Quizzen leveres **som den er** uden nogen form for garanti, hverken direkte eller indirekte.
- Brug af quizzen sker på eget ansvar.

## Brugsbetingelser
- Quizzen må kun anvendes til **ikke-kommerciel brug**.
- Quizzen må anvendes til privat eller undervisningsmæssig brug.
- Ingen del af quizzen må opfattes som officiel undervisning eller certificering.
- Ved at bruge quizzen accepterer du, at udvikleren ikke kan holdes ansvarlig for fejl, mangler eller misforståelser.
